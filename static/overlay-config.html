A free stats overlay for your MKWC broadcast: a live head-to-head of any two teams (win %, average, point difference, consistency), refreshed automatically as scores come in. Pick the two teams, copy the URL, and drop it into an OBS browser source.

3 · Live preview

Shown on a checkerboard so you can see the transparent background. In OBS it sits over your scene.

4 · Add it to OBS

  1. In OBS, add a source → Browser.
  2. Paste the URL above into the URL field.
  3. Set width 720, height 360 (adjust to taste).
  4. Tick Shutdown source when not visible and Refresh when active if you like.
  5. Position it on your scene — the background is transparent.
